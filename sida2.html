Sida2 Om CSS

  1. Vad är CSS?
  2. CSS, eller Cascading Style Sheets, är ett formatmallsspråk som används för att beskriva utdata från dokument skrivna i ett märkningsspråk som HTML. CSS tillåter webbutvecklare att kontrollera stil, färger, avstånd och andra aspekter av webbsidors utseende. (styling) används för att dekorera och strukturera webbplatsen.
  4. Varför finns CSS?
  5. Vi använder css för att dekorera hemsidan, för att förbättra hemsidan i dess svaga områden. Vi använder css på många ställen på en webbplats för att göra den mer läsbar eller inte tråkig i hur webbplatsen ser ut.
  6. Hur använder vi oss av CSS?
  7. Vi använder css i olika stilar, en av dem är att använda stiltaggen som vi skriver i html-stil style och vi säger enter style och vi väljer elementet som vi vill ändra dess utseende genom att använda attributet för att skapa varje sektion eller sektioner.
  8. Det finns olika versioner av CSS:
  10. Vilka versioner finns det?
  11. CSS1, CSS2, CSS3, och CSS4
  12. Varför finns det olika versioner? (Varför är det inte bara en)
  13. eftersom css blir bättre och bättre och nya funktioner kommer ut för att göra den mer attraktiv och kraftfull. för att göra skillnad i hur webbplatsen ser ut
  14. Varför är det viktigt att veta vilken version som en viss CSS-egenskap tillhör? (Kan detta påverka webbplatsen?)
  15. Det är viktigt att veta vilken kategori funktionen tillhör eftersom den nya funktionen kanske inte stöds i äldre webbläsare. Och vi vet hur man får hemsidan att se ut
